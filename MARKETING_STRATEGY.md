- **Объём рынка:** $2.14 млрд (2025) — крупнейший в Центральной Азии

- **104 фармацевтических проекта** на сумму $1.25 млрд до 2026 года

### Глобальный рынок чистых помещений
- **Объём:** $8.8 млрд (2024) → $12.3 млрд (2029)
- **Фарма-сегмент:** 48% рынка
- **Расходные материалы:** 55% продаж

## Почему контекстная реклама не работает

| Проблема | Причина |
|----------|---------|
| Нецелевой трафик | Запросы "перчатки", "маски", "халаты" приводят обычных потребителей |
| Высокая стоимость клика | Конкуренция с B2C-продавцами медтоваров |
| Низкая конверсия | Посетители ищут бытовые товары, а не GMP-продукцию |
| Длинный цикл сделки | B2B-клиенты не покупают по первому клику |

## Стратегические направления

### 1. ЭКСПЕРТНЫЙ КОНТЕНТ-МАРКЕТИНГ

#### Образовательный контент
- **Гайды по стандартам:** ISO 14644, GMP, Annex 1 EU GMP
- **Сравнительные обзоры:** материалы, классы чистоты, бренды
- **Калькуляторы:** расчёт потребности в расходниках
- **Чек-листы:** аудит чистых помещений, подготовка к проверкам

#### Форматы контента
| Формат | Цель | Частота |
|--------|------|---------|
| Экспертные статьи | SEO + авторитет | 2-4/месяц |
| Технические спецификации | Конверсия | По мере необходимости |
| Кейсы клиентов | Доверие | 1-2/квартал |
| Видео-обзоры продукции | Вовлечение | 2-4/месяц |
| Вебинары | Лидогенерация | 1/квартал |

#### Темы для контента
1. "Как выбрать одежду для чистых помещений по классу ISO"
2. "Требования GMP к расходным материалам: полное руководство"
3. "Сравнение нитриловых и латексных перчаток для фармы"
4. "Новые требования Annex 1 EU GMP: что изменилось"
5. "Контроль частиц в производстве: методы и оборудование"
6. "Валидация уборки в чистых помещениях: пошаговый гайд"

### 2. LINKEDIN-СТРАТЕГИЯ (Приоритет #1)

LinkedIn — главная платформа для B2B в промышленности.

#### Корпоративная страница
- **Частота постов:** 2-3 раза в неделю
- **Контент-микс:**
  - 40% — образовательный контент
  - 30% — новости отрасли и компании
  - 20% — кейсы и отзывы
  - 10% — команда и культура

#### Типы постов
1. **Карусели:** "5 ошибок при выборе перчаток для класса ISO 5"
2. **Видео:** краткие обзоры продукции (30-60 сек)
3. **Опросы:** вовлечение аудитории
4. **Документы:** чек-листы и гайды для скачивания

#### Employee Advocacy (Адвокация сотрудников)
- Оптимизировать профили ключевых сотрудников
- Директор, менеджеры — публикуют экспертный контент
- Расширение охвата через личные сети

#### Целевые должности для таргетинга
- Директор по производству
- Главный технолог
- Начальник ОКК (отдел контроля качества)
- Инженер по чистым помещениям
- Специалист по закупкам

#### Целевые отрасли
- Фармацевтика
- Биотехнологии
- Медицинские изделия
- Электроника и полупроводники
- Пищевая промышленность

| "дезинфектант" | "дезинфектант для чистых помещений класса A" |

#### Целевые запросы для SEO
- "оборудование для чистых помещений Узбекистан"

- "одежда для фармацевтического производства"
- "ISO 14644 требования к одежде"
- "валидация чистых помещений"
- "расходники для чистых комнат"

#### Технический SEO

- Schema.org разметка для продуктов

   - KRKA
   - STADA
   - Yuria-Pharm

3. **Новые проекты (строящиеся заводы)**
   - Резиденты Tashkent Pharma Park

#### ABM-подход
1. **Исследование** — изучение компании, её проектов, ЛПР
2. **Персонализированный контент** — кейс под специфику клиента
3. **Multi-touch кампания** — LinkedIn + email + звонок + встреча
4. **Долгосрочное выращивание** — nurturing до готовности к покупке

## KPI и метрики

### Лидогенерация
- Количество квалифицированных лидов в месяц
- Стоимость привлечения лида (CPL)
- Конверсия лид → клиент

### Контент
- Органический трафик на сайт
- Время на сайте
- Скачивания материалов
- Подписки на рассылку

### LinkedIn
- Рост подписчиков
- Engagement rate
- Клики на сайт
- InMail response rate

### Выставки
- Количество контактов
- Качество контактов (оценка)
- Сделки после мероприятия

## Ключевые сообщения бренда

### Позиционирование

5. **Техническая поддержка** — консультации по подбору и применению
